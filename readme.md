# Rust Piano

This is a terminal-based program written in the Rust programming language. Once the program is ran, you will be given a display of all piano notes correlated to your keyboard's keys. Once you press these keys, your machine will produce a audible piano note.

## Instructions for Build and Use

Steps to build and/or run the software:

1. You will need to make sure cargo Rust is downloaded and installed on your machine.

2. Once have cargo Rust installed, download the zip file for this repo.

3. Then once you have the zip file, extract the zip file, click the 'src' file and open it with your machine's terminal, then run the 'src' file with 'cargo build' & 'cargo run'.

4. Then the program will run inside the user's terminal.

Instructions for using the software:

1. The keys: 'a', 's', 'd', 'f', 'g', 'h', and 'j' will play base notes. You tap these keys and base piano notes will play.
2. The keys: 'w', 'e', 't', 'y', 'u', and 'i' will play sharp notes. 
3. Because of the threading in the program, multiple notes can be played at the same time.
